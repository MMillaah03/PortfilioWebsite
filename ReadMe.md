# Portfolio Website

Welcome to my personal portfolio website repository. <br> This project showcases my skills, projects, and experience as a developer through a personal portfolio website.

## 🌐 Overview

This is a simple responsive portfolio website designed to present information about me in a clean and organized way.  
It serves as a central hub for my projects, skills, and professional background.

## 📁 Project Structure

```text
portfolio/
│
├── css/ # Stylesheets for the website
├── pages/ # Additional HTML pages (About, Projects, etc.)
├── assets/ # Images, icons, and media files
├── index.html # Main landing page
```


## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design principles
